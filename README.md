# Property Walkthrough Checklist — Tooensure LLC

A field-ready property preservation inspection app for tracking walkthrough stops, checklist items, photos, and generating reports.

## 🌐 Live Site

**[https://shawndelainebellazanloop.github.io/Property-Preservation/](https://shawndelainebellazanloop.github.io/Property-Preservation/)**

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. (Optional) Set `GEMINI_API_KEY` in `.env.local` for AI features
3. Run the app:
   `npm run dev`
4. Open: `http://localhost:3000`

## Deploy

Pushes to `main` automatically deploy to GitHub Pages via the Actions workflow.
